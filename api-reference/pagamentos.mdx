## Criar Pagamento PIX

Cria uma nova transação PIX e gera o QR Code e o código Copia e Cola.

<RequestExample>
  ```bash POST /payment/create
  curl -X POST 'https://api.zynxwallet.com.br/api/v1/payment/create' \
    -H 'X-API-Key: zynx_sua_chave_aqui' \
    -H 'Content-Type: application/json' \
    -d '{
      "value": 150.00,
      "description": "Venda de Produto #123",
      "coverFee": true
    }'
  ```
</RequestExample>

### Parâmetros do Corpo

<ResponseField name="value" type="number" required>
  Valor em reais (ex: 100.50). Mínimo R$ 1,00.
</ResponseField>
<ResponseField name="description" type="string">
  Descrição que aparecerá no extrato.
</ResponseField>
<ResponseField name="coverFee" type="boolean">
  Se `true`, a taxa é adicionada ao valor total. Se `false`, a taxa é descontada do valor recebido.
</ResponseField>
<ResponseField name="splitUser" type="string">
  Email de outro usuário Zynx para divisão de valores.
</ResponseField>
<ResponseField name="splitTax" type="number">
  Porcentagem (0.01 a 100) para o `splitUser`.
</ResponseField>

### Resposta

<ResponseExample>
  ```json 201
  {
    "success": true,
    "data": {
      "id": "pay_abc123",
      "transactionId": "mistic_xyz789",
      "valueInReais": 153.00,
      "netValue": 15000,
      "fee": 300,
      "status": "PENDING",
      "qrcodeUrl": "data:image/png;base64,...",
      "copyPaste": "00020126360014br.gov.bcb.brcode...",
      "createdAt": "2024-02-16T15:00:00Z"
    }
  }
  ```
</ResponseExample>

## Consultar Status de Pagamento

Verifica os detalhes e o status atual de um pagamento específico.

<RequestExample>
  ```bash GET /payment/get/{id}
  curl 'https://api.zynxwallet.com.br/api/v1/payment/get/pay_abc123' \
    -H 'X-API-Key: zynx_sua_chave_aqui'
  ```
</RequestExample>

### Parâmetros da URL

<ResponseField name="id" type="string" required>
  O ID do pagamento retornado na criação.
</ResponseField>

### Resposta

<ResponseExample>
  ```json 200
  {
    "success": true,
    "data": {
      "id": "pay_abc123",
      "status": "COMPLETED",
      "value": 15300,
      "netValue": 15000,
      "updatedAt": "2024-02-16T15:05:00Z"
    }
  }
  ```
</ResponseExample>

## Listar Pagamentos

Retorna uma lista paginada de todos os seus pagamentos.

<RequestExample>
  ```bash GET /payment/list
  curl 'https://api.zynxwallet.com.br/api/v1/payment/list?status=COMPLETED&limit=10' \
    -H 'X-API-Key: zynx_sua_chave_aqui'
  ```
</RequestExample>

### Parâmetros da Query

<ResponseField name="status" type="string">
  Filtrar por status (`PENDING`, `COMPLETED`, `FAILED`).
</ResponseField>
<ResponseField name="limit" type="integer">
  Quantidade por página (padrão 50, máx 1000).
</ResponseField>
<ResponseField name="offset" type="integer">
  Pular registros (padrão 0).
</ResponseField>

### Resposta

<ResponseExample>
  ```json 200
  {
    "success": true,
    "data": [
      {
        "id": "pay_abc123",
        "status": "COMPLETED",
        "value": 15300,
        "netValue": 15000,
        "updatedAt": "2024-02-16T15:05:00Z"
      }
    ]
  }
  ```
</ResponseExample>
