## Solicitar Saque PIX

Transfere saldo da sua conta Zynx para uma chave PIX externa.

<RequestExample>
  ```bash POST /withdraw/create
  curl -X POST 'https://api.zynxwallet.com.br/api/v1/withdraw/create' \
    -H 'X-API-Key: zynx_sua_chave_aqui' \
    -H 'Content-Type: application/json' \
    -d '{
      "amount": 500.00,
      "pixKey": "12345678900",
      "pixKeyType": "CPF",
      "description": "Retirada de Lucro"
    }'
  ```
</RequestExample>

### Parâmetros do Corpo

<ResponseField name="amount" type="number" required>
  Valor em reais.
</ResponseField>
<ResponseField name="pixKey" type="string" required>
  Chave PIX de destino.
</ResponseField>
<ResponseField name="pixKeyType" type="string" required>
  `CPF`, `CNPJ`, `EMAIL`, `PHONE` ou `RANDOM`.
</ResponseField>
<ResponseField name="description" type="string">
  Descrição da retirada.
</ResponseField>

### Resposta

<ResponseExample>
  ```json 200
  {
    "success": true,
    "message": "Saque solicitado com sucesso"
  }
  ```
</ResponseExample>

## Enviar Dinheiro (Reembolso/Transferência)

Realiza um envio direto de valores. Ideal para automação de reembolsos.

<RequestExample>
  ```bash POST /payment/send
  curl -X POST 'https://api.zynxwallet.com.br/api/v1/payment/send' \
    -H 'X-API-Key: zynx_sua_chave_aqui' \
    -H 'Content-Type: application/json' \
    -d '{
      "amount": 25.50,
      "pixKey": "cliente@email.com",
      "pixKeyType": "EMAIL",
      "description": "Reembolso Pedido #99"
    }'
  ```
</RequestExample>

### Parâmetros do Corpo

<ResponseField name="amount" type="number" required>
  Valor em reais.
</ResponseField>
<ResponseField name="pixKey" type="string" required>
  Chave PIX de destino.
</ResponseField>
<ResponseField name="pixKeyType" type="string" required>
  `CPF`, `CNPJ`, `EMAIL`, `PHONE` ou `RANDOM`.
</ResponseField>
<ResponseField name="description" type="string">
  Descrição da transferência.
</ResponseField>

### Resposta

<ResponseExample>
  ```json 200
  {
    "success": true,
    "message": "Transferência realizada com sucesso"
  }
  ```
</ResponseExample>
